Privacy Policy

This page describes how StvFriends collects, stores, and uses data retrieved from Strava.

1. Data Collected

Only the following authorized Strava data is collected:

  • Activity type
  • Date
  • Distance
  • Moving time
  • Activity ID

2. Data Storage

Data is stored securely and used only for the duration of the private challenge.

3. Data Usage

  • To calculate rankings
  • To generate challenge statistics

4. Data Security

Secure storage practices are followed, and access is restricted to project administrators.

5. Removal Request

You may request removal of your data at any time by emailing:
diego.reports@gmail.com

6. Third Parties

The application connects only to Strava’s official API, using OAuth.
