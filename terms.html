Termos de Uso – StravaFriends

Ao utilizar o StravaFriends, você concorda com os termos descritos abaixo.

1. Finalidade

O objetivo do projeto é fornecer uma competição recreativa entre amigos usando dados esportivos do Strava.

2. Responsabilidade do Usuário

O usuário autoriza acesso às suas atividades esportivas via Strava OAuth.

3. Limitações

O projeto não possui fins comerciais. O uso é limitado ao período da competição.

4. Encerramento

Os dados podem ser excluídos a pedido do usuário a qualquer momento.
